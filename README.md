# Queue (Expo)

Expo app for the QuickFit migration target.

## Setup

```bash
bun install
```

## Run

- Start Expo:
  - `bun run start`
- Android (native Windows emulator flow):
  - `bun run android`
- Android doctor:
  - `bun run android:doctor`
- Web:
  - `bun run web`
- Lint:
  - `bun run lint`

## Notes

- Android workflow is Windows-first (no WSL requirement).
- See `docs/android-windows-setup.md` for details.
